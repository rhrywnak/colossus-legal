# Colossus-Legal (Skeleton)

This is a project skeleton for the Colossus-Legal application.

- Rust backend (Axum) under `backend/`
- React + Vite + TypeScript frontend under `frontend/`
- Shared Neo4j instance configured via environment variables
- `docs/` and `prompts/` are placeholders; copy in your architecture docs and prompt files.

Next steps:

2. Configure Neo4j env vars for your shared instance.
